## 🪟 What this app does

claude-code-md-hook helps Claude Code read common document files as markdown.

It can work with:

- PDF
- DOCX
- XLSX
- PPTX
- HTML

The app converts these files before Claude Code reads them. This helps keep file reading light and reduces extra token use from image-based reading. It also makes large files easier to scan.
